# Mock Issue Cycle

This folder contains a complete end-to-end sample issue lifecycle using the same
artifact model used by the app.

Path:

- `quick-deal/Issues/ISS-900001`

Cycle stages:

1. Submitted as `new`
2. Clarified and moved to `ready_for_fix`
3. Started by dev (`in_progress`)
4. Delivered fix (`fixed_pending_verify`)
5. Verified by QA (`verified_closed`)
